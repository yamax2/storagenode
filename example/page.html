Storage Node Usage Example

Cross-Origin Session Flow

This page demonstrates loading images from a Storage Node (https://node1.localhost:8443) hosted on a different domain than this example app (https://application.localhost:8443/).

Step 1: Server generates a JWT token with claims: method=GET, uri=/service/start
Step 2: Browser calls /service/start with the JWT in X-Session-Token header
Step 3: Storage Node validates JWT and sets storagesession cookie
Step 4: Images load from /data/* using the session cookie
